### Installation 
- git clone `git clone https://github.com/nati2323/5-math-magicians-full-website` .
- Type `cd 5-math-magicians-full-website`.

#### Install dependencies
```
npm install
```
### Build the app
```
npm run build
```
### Run the app

```
npm start
```
